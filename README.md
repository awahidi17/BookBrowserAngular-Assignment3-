# Book Browser - Assignment 3

Angular standalone application that manages a list of books.
- Angular routing
- HttpClient CRUD service
- angular-in-memory-web-api simulated backend
- Angular signals for selected book and favorites
- Search, sorting, favorite-only toggle, and localStorage favorites

## Features
- View list of books
- View book details
- Add new books
- Edit existing books
- Delete books
- Local images for book covers
- Responsive UI

## Run the Project
npm install
ng serve
